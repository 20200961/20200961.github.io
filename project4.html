프로젝트 소개

Creator-Flex는 크리에이터와 광고주 간의 협업을 투명하고 효율적으로 관리하기 위한 MCN(Multi-Channel Network) 비즈니스 올인원 플랫폼입니다.


단순한 연결을 넘어 계약 체결, 광고 집행, 실시간 소통, 법률 준수 확인까지 프로젝트의 전 과정을 통합 관리할 수 있는 유연한 업무 환경을 제공합니다.

주요 기능

1
CRUD를 활용한 MCN 직원 관리 시스템
2
SSE를 활용한 실시간 알림 시스템
3
JWT 기반 권한 관리 서비스로 역할별 기능 접근 제어
4
OpenAI 연동을 통한 데이터 분석 및 스마트 업무 지원 서비스
5
AWS S3 파일 백업 및 Redis 캐싱을 통한 빠른 데이터 처리 성능 확보

주요 기여

- 프론트엔드/백엔드를 구분하지 않고 전반적인 웹 개발 역량 향상을 목표로 한 팀 단위 프로젝트 수행


- SSE(Server-Sent Events)를 활용한 실시간 알림 구현


- CRUD를 활용한 광고, 계약, 크리에이터 관리 시스템 구현


- 다차원 필터링 기술이 적용


- Spring Boot와 React 간의 효율적인 API 명세 설계 및 데이터베이스 정규화 수행

기술 스택
